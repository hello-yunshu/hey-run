# 嘿，跑～

这是 云云舒 的个人博客，域名嘛，就是 [hey.run](https://hey.run) 啦。

基于 Hugo + Blowfish 主题搭建，通过 GitHub Actions 自动部署到 GitHub Pages。

博客里主要写些服务器运维、网络技术、编程开发之类的东西，偶尔也有些乱七八糟的碎碎念～

## 本地跑起来

```bash
git clone --recursive https://github.com/hello-yunshu/hey-run.git
cd hey-run
hugo server
```

然后浏览器打开 `http://localhost:1313` 就行了。
