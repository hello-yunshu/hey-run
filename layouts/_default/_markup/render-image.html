{{- define "RenderImageSimple" -}} {{- $imgObj := .imgObj -}} {{- $src := .src -}} {{- $alt := .alt -}} {{- end -}} {{- define "RenderImageResponsive" -}} {{- $imgObj := .imgObj -}} {{- $alt := .alt -}} {{- $originalWidth := $imgObj.Width -}} {{- $quality := site.Params.imageQuality | default 72 -}} {{- $img800Width := cond (gt $originalWidth 800) 800 $originalWidth -}} {{- $img800 := $imgObj.Resize (printf "%dx webp q%d" $img800Width $quality) -}} {{- $img1280 := $img800 -}} {{- if gt $originalWidth 1280 -}} {{- $img1280 = $imgObj.Resize (printf "1280x webp q%d" $quality) -}} {{- end -}} {{- $srcset := printf "%s %dw, %s %dw" $img800.RelPermalink $img800.Width $img1280.RelPermalink $img1280.Width -}} {{- end -}} {{- define "RenderImageCaption" -}} {{- with .caption -}} {{- end -}} {{- end -}} {{- $disableImageOptimizationMD := .Page.Site.Params.disableImageOptimizationMD | default false -}} {{- $urlStr := .Destination | safeURL -}} {{- $url := urls.Parse $urlStr -}} {{- $altText := .Text -}} {{- $caption := .Title -}} {{- $isRemote := findRE "^(https?|data)" $url.Scheme -}} {{- $resource := "" -}} {{- if not $isRemote -}} {{- $resource = or ($.Page.Resources.GetMatch $urlStr) (resources.Get $urlStr) -}} {{- end -}}
