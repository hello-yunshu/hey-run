> 旧文归档：本文整理的是当时的教育优惠入口和验证方式。Apple、微软、JetBrains、GitHub、AWS 以及云厂商的资格规则和活动额度经常变化，申请前请以各平台官网当前页面为准。

其实很多时候，我们有身边有很多资源可以利用，但大多是都是被我们忽视，或者完全不知情。

所谓教育优惠，就是一项针对于在校大学生和教职员工推出的特殊优惠活动。一些公司会将旗下产品或服务以一定的折扣，甚至免费提供给高校师生。

申请教育优惠一般有一个前提，那就是要有一个学校提供的「校园邮箱」。在国内，也就是 @xxx.edu.cn 结尾的邮箱。

当然，如果你没有校园邮箱，也不要伤心。本文中的大部分教育优惠也是可以通过上传学生证验证的，只是相比起来会复杂一点。

#### Apple 苹果教育商店

* * *

>> 认证方式：学生证

苹果针对大学生，被大学录取的学生，为大学生购买产品的家长、教师、家庭教师和各级教职员工提供教育优惠。目前，所有 Mac 机型， iPad 机型，和 AppleCare 产品均可享受优惠。苹果的教育优惠产品可以通过线上渠道和线下渠道两种方式购买，具体如下：

  * 线上渠道：apple.com 教育商店，以及天猫零售店。
  * 线下渠道：Apple Store 零售店。

如果你所在的地区有 Apple Store 零售店，线下渠道是首选的购买途径。你只需要拿着学生证去就好了，非常方便。注意，这里的 Apple Store 零售店并不是指更广泛分布的授权经销商，具体可以到[此页面](https://www.apple.com/cn/retail/storelist/)查找中国地区的零售店列表。

如果你附近没有 Apple Store 零售店，可以通过官网或者 Apple 天猫旗舰店验证学生身份购买。我自己体验过在官网购买，虽然天猫零售店显示支持教育优惠，但是我没有找到入口。所以，还是推荐大家在官网购买。通过官网教育商店购买的流程大致如下：

![](/images/posts/zai-xiao-shi-sheng-fu-li-he-ji-apple-wei-ruan-autodeskdeng-jiao-yu-you-hui-zhi-nan/cover.avif)

打开[苹果官网](https://www.apple.com/cn/)，下拉网页到底部，找到「高校师生选购」选项。

点击该选项进入教育商店，你可以看到网站顶部有「教育商店商店首页」的字样。

此时，页面中的所有商品均显示教育优惠后的价格，然后完成正常下单流程。

当你提交订单后，苹果会向你的邮箱发送一封验证学生身份的邮件。按照邮件说明，将学生证封面、内容页、注册详情页拍照并发送到提供的指定邮箱即可。

目前，苹果教育优惠幅度和产品售价成正比。也就是说售价越高的产品，往往优惠的力度也就越大。另外，每年的秋季开学前，苹果一般会推出更大力度的学生优惠活动。

#### 阿里云 & 腾讯云主机

* * *

>> 认证方式：学信网 & 银行卡

国内的阿里云和腾讯云都针对其云产品提供了教育优惠，你可以以极低的价格获得一台云主机。

  * 阿里云的教育优惠又称为「[云翼计划](https://promotion.aliyun.com/ntms/campus2017.html)」，只需要 9.9 元/月即可拥有一台原价 111.1 元/月的云服务器 ECS。
  * 腾讯云的教育优惠又称为「[云+校园](https://www.qcloud.com/act/campus)」，一台学生机的价格为 10 元/月。与阿里云的学生机不同的是，腾讯云学生机是每天上午 9 点限量抢购。

获取两者的方式差不多，无需校园邮箱，通过学信网学籍信息验证即可。打开设置页面，找到学生认证，然后按照你在学信网的信息填写，提交后会实时返回验证结果。

阿里云和腾讯云配置方面差不多，默认情况下，阿里云为 1核 CPU 以及 2GB 内存，腾讯云为 1核 CPU 以及 1GB 内存。具体信息，你可以到相应的网页了解。这两种云主机虽然便宜，但都有着相似的弱点：

  * 地区只能选在国内的数据中心（腾讯云未亲自测试），无法搭建科学上网服务。
  * 搭建网站时，若需绑定域名，需要先进行域名备案（域名备案时间跨度非常长，paniy亲测），否则只能通过 IP 地址访问。

#### AWS 信用额度

* * *

>> 认证方式：校园邮箱

AWS 是亚马逊云服务的简称，它也是全世界最出色的云服务之一。亚马逊的 EC2 主机和 S3 可拓展储存被大量的互联网公司使用，其中就包括 GitHub。

如果你之前对 AWS 有所了解，你或许知道 AWS 针对新用户提供了 1 年期的[免费套餐](https://aws.amazon.com/cn/free/)，这里面包括一台基础版本的 Amazon EC2 云主机。但是，这台云主机有每月 15 GB 的流量限制，超出这个限制就会自动扣费。免费工具包中提供的 $75-$150 信用额度可以抵消掉这笔费用以及其他的费用。下面，就一步一步带你申请这笔信用额度。

![](/images/posts/zai-xiao-shi-sheng-fu-li-he-ji-apple-wei-ruan-autodeskdeng-jiao-yu-you-hui-zhi-nan/09.avif)

首先，在 GitHub 学生开发工具包页面找到 AWS Educate 一项，点击唯一的 unique link 进入如下所示的 AWS Educate 申请页面。

![](/images/posts/zai-xiao-shi-sheng-fu-li-he-ji-apple-wei-ruan-autodeskdeng-jiao-yu-you-hui-zhi-nan/10.avif)

在这个页面里，有这几项需要注意：

  * Email 地址。这里必须填写你的 @xxx.edu.cn 校园邮箱，不然无法通过。
  * AWS Account ID。填写此对话框前，你需要提前到 AWS 使用任意邮箱注册一个有效账号。注意，这里不是填写你的 AWS 注册邮箱，而是到账户「设置页面」找到自己对于的 12 位数字 ID。
  * Promo 兑换码。这一项再通过 unique link 跳转过来时会自动填好，如果为空，则无法获得信用额度。你可以重新回到 GitHub 页面，点击唯一链接跳转。

填写完表单，AWS 会验证你的校园邮箱。

验证完毕之后，你的校园邮箱会收到一封标题为「AWS Educate Application Approved」的信用额度批准邮件，里面会有信用额度的兑换码。

此邮件内容较多，你需要仔细阅读才能找到包含信用额度兑换码的 Credit Code 字段。

![](/images/posts/zai-xiao-shi-sheng-fu-li-he-ji-apple-wei-ruan-autodeskdeng-jiao-yu-you-hui-zhi-nan/11.avif)

到目前为止，就差最后一步了。你需要登陆你提前注册好的的 AWS 账号。

在控制面板左侧找到「服务抵扣金额」一项，输入兑换码即可完成信用额度注入。

虽然，GitHub 上面写着，这项信用额度会在 $75-$150 之间，但是我申请后直接给了 $150 的最大额度。

AWS 的信用额度是实打实的真金白银，所以申请过程稍微繁琐。当然，如果你正在使用 AWS，我相信你是不会在意过程的。

#### AutoDesk 教育社区

* * *

>>认证方式：声明学生身份

Autodesk 是一家在三维设计、工程设计及娱乐软件方面处于世界领先的公司。旗下的 AutoCAD，Maya 以及 3ds Max 被许多专业的学生使用。Autodesk 几乎所有的软件都提供完全免费的教育版本，单次授权期为 3 年。

打开 [Autodesk 教育社区页面](https://www.autodesk.com.cn/education/free-software/all)，选择你想下载的软件。进入下一个页面后，点击创建账户并注册。完成邮件地址激活后，就进入如下图所示的教育信息录入表单。

![](/images/posts/zai-xiao-shi-sheng-fu-li-he-ji-apple-wei-ruan-autodeskdeng-jiao-yu-you-hui-zhi-nan/12.avif)

按照真实情况填入自己的学校等信息（图一）。如果你的学校无法选择（图二），就点击「找不到自己的学校？」，然后添加一个教育机构（图三）。经过测试，找不到的概率还是蛮大的。

Autodesk 的教育优惠申请是我见过最简单的，无需任何证明材料。当你填完上述表单时，你就已经获得了优惠资格，免费的兑换码会直接发送到你的邮箱。
