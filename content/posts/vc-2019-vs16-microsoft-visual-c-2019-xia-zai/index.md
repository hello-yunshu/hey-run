前段时间，微软发布了最新的Visual Studio 2019 (以下简称VS)，笔者并不怎么使用VS平台，上次使用是为了开发Microsoft Store应用而用了一段时间。当时用的还是VS 2017，没想到现在这么快就到了VS 2019（VS16）。

![](/images/wp-content/uploads/2019/05/20190525210630-1024x343.jpg)

> Microsoft Visual Studio是VS的全称。VS是美国微软公司的开发工具包系列产品。VS是一个基本完整的开发工具集，它包括了整个软件生命周期中所需要的大部分工具，如UML工具、代码管控工具、集成开发环境(IDE)等等。所写的目标代码适用于微软支持的所有平台，包括Microsoft Windows、Windows Mobile、Windows CE、.NET Framework、.Net Core、.NET Compact Framework和Microsoft Silverlight 及Windows Phone。

既然有了最新的VS 2019，那么自然也会有Visual C++ Redistributable for Visual Studio 2019 (version 16.0)。一般情况下，程序猿不会如此激进的使用高版本的VS，但是对于非生产环境诸如GitHub上的部分程序就不好说了。

网上关于Microsoft Visual C++ 2019的资料并不多，笔者也是找了半天找不到。最后，笔者居然发现Microsoft Visual C++ 2019就安静的躺在Visual Studio官网里面。授人以鱼不如授人以渔，以下就教大家如何在官网下载Microsoft Visual C++ 2019。

## 官网下载Microsoft Visual C++ 2019

首先我们需要一个微软的账号，然后进入Visual Studio的官网：[my.visualstudio.com](<https://my.visualstudio.com/>)，根据提示，登录后进入以下页面。

![](/images/wp-content/uploads/2019/05/20190525202144-1024x482.jpg)

点击上方下载，在下载页面找到Visual Studio 2019并点击。

![](/images/wp-content/uploads/2019/05/20190525202330-1024x482.jpg)

进入Visual Studio 2019的下载页面，往下寻找就可以找到需要的Visual C++ Redistributable for Visual Studio 2019 (version 16.0)

![](/images/wp-content/uploads/2019/05/20190525202808-1024x478.jpg)

根据自己的操作系统选择是32位还是64位亦或者是amd版本。选择自己的语言(实际意义不大)，最后，在点击Download即可。就是这么简单。

安装后在程序和功能中就可以看到安装的依赖包。

![](/images/wp-content/uploads/2019/05/20190525205421-1024x653.jpg)

从程序的名字Microsoft Visual C++ 2015-2019 Redistributable可以看出，现在微软似乎把VS依赖包作为一种长期更新的软件。也就是说不会有之前类似Microsoft Visual C++ 2010的包了。对于我们用户来说，由于依赖包没有自动更新功能，多少有些麻烦。

当然还有一个下载的办法，在[Microsoft Visual Studio下载](<https://www.visualstudio.com/downloads/>)页面中，进入“其他工具和框架”，便可找到下载地址。
